Преимущества и недостатки Python
Плюсы:
Простота синтаксиса. Python был придуман как простой язык для превлечения внимания еще с самого начала
Большая популярность. если не самый, то Python один из самых популярных языков программирования
Активное сообщество. Для языка существует много библиотек для любой задачи, по каждой есть много уроков на Youtubе
Наличие поддержки ООП
Легкость в обучении для новичков. Многое, для чего в других языках требуются особые навыки и знания, можно сократить, преведя в более понятный и минималистичный вид
Минусы:
Очень медленная работа. В некоторых проектах, использование иных языков необходимо для оптимизации скорости работы
Большие затраты памяти. Для облегчения написания кода Python тратит много памяти
Небольшое кол-во возможностей для оптимизации кода. Особенно после второй версии, когда убрали некоторые пути реализации задач, на Python мало путей устранения предыдущих двух пунктов

Синтаксис:
Python известен своим минималистичным и, на момент выхода, иновационным синтаксисом. Кого-то он отталкнет, а кого-то превлечет, и особенно это работает на новичков. Избавление от ";" и любых обозначений тел условий или циклов кроме привычных отступов уже привычно для программистов на Python, но некоторым удобней видеть символьные обозначения действий. Радикально, язык не отличается от других, а иногда даже наоборот пытается забрать многое у других популярных языков программирования

Скорость печати кода:
Так же язык отличается количеством кода и строк нужных для выполнения задач. Когда человек пишет одну строку на Python, это могут быть несколько строк на другом языке, и это не плюс, так как другие языки помогают раскрыть процесс, и детально настроить нужные подпроцессы. Особый эффект демонстрируется на новичках, которые видят сокрощенность кода и остаются под большим впечатлением

Динамическая типизация:
Динамическая типизация – приём, используемый в языках программирования, при котором переменная связывается с типом в момент присваивания значения, а не в момент объявления переменной. Другими словами, в большинстве языков тип переменной нельзя менять, и в некоторых ситуациях нужно создавать больше переменных, но Python такое не грозит. В любой строке кода, даже без специальных операторов и функций, можно поменять тип переменной, что вместе со всеми пунктами делает код на Python максимально близким к речи носиталя английского языка
